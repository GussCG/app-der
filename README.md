# APP-DER - Editor de Diagramas E-R

Aplicación web para el diseño interactivo de diagramas Entidad-Relación con validación automática y generación de modelo relacional y código SQL.

Integra asistencia mediante IA (Gemini API).

## Stack Tecnológico

React 19, Vite, React Flow / XYFlow, MUI, ELKJS (auto layout), Gemini API, html-to-image, DownloadJS

## Funcionalidades

- Creación visual de entidades y relaciones
- Validación de integridad referencial
- Generación automática de modelo relacional
- Exportación a SQL
- Exportación como imagen
- Auto layout con ELKJS
- Asistencia mediante IA para diseño y dudas técnicas

## Enfoque Técnico

- Arquitectura basada en componentes
- Manejo avanzado de estado en diagramas
- Validación estructural de reglas E-R
- Generación automática de código SQL
